Handmade Web Starter Kit

마크다운으로 글을 쓰고, 바로 HTML 레이아웃에서 확인할 수 있는 극간결 스타터 키트입니다. 필요한 것은 텍스트 에디터뿐입니다.

워크스페이스로 이동 자체 렌더러 · 기본 디자인 토큰

1. 입력

Markdown 작성

좌측에 마크다운을 입력하면 우측 미리보기에서 즉시 렌더링됩니다. 글쓰기 흐름을 깨지 않는 최소한의 인터랙션만 포함했습니다.

2. 미리보기

HTML 프리뷰

스타터 디자인 시스템이 적용된 결과물을 확인하세요. CSS 변수만으로 색상, 간격, 서체를 조정할 수 있습니다.

필요한 부분만 추가해도 디자인이 무너지지 않도록 설계했습니다. 자바스크립트가 동작하지 않아도 원본 마크다운을 그대로 확인할 수 있습니다.
