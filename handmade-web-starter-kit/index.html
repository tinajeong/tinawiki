Handmade Web Starter Kit

심플한 블로그 시동 패키지

프린트 감성과 타이포그래피 중심 레이아웃으로 시작하는 최소한의 블로그 템플릿입니다. Markdown-it을 바로 가져와 글을 렌더링할 수 있습니다.

구성

기본 레이아웃

헤더, 본문, 보조 패널만으로 구성된 단일 칼럼 레이아웃입니다. HTML 만으로도 읽을 수 있고, JS는 Markdown 렌더링만 담당합니다.

  • 타입 스케일: 루트 폰트 사이즈와 간단한 모듈러 스케일.
  • 색상 토큰: 배경, 텍스트, 액센트만 정의해 유지보수 단순화.
  • 구문 강조 없음: 필요 시 확장할 수 있도록 최소한만 제공합니다.

렌더링 예시

JS를 끈 상태에서는 아래 Markdown을 그대로 보여주고, 켜두면 즉시 HTML로 변환합니다.

# Markdown으로 작성하는 첫 글

- 마크다운-it CDN을 바로 불러옵니다.
- 스크립트는 `data-markdown-source` 블록을 찾아 HTML로 변환합니다.
- CSS는 읽기 경험을 해치지 않는 기본 토큰만 제공합니다.

[코드를 참고해 바로 확장해 보세요.](./starter.js)
